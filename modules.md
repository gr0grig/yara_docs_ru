# Модули

Модули — это способ, предоставляемый YARA для расширения своих возможностей. Они позволяют определять структуры данных и функции, которые можно использовать в правилах для формулирования более сложных условий. Здесь описаны некоторые модули, официально распространяемые с YARA. Вы также можете научиться создавать собственные модули в разделе [Написание собственных модулей](writingmodules.md).
