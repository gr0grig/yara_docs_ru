# Модуль String

*Добавлен в версии 4.3.0*

Модуль String предоставляет функции для работы со строками, возвращаемыми модулями. Строки, упоминаемые здесь, — это не YARA-строки, определённые в секции strings вашего правила.

## Справочник

### to_int(string)

*Добавлен в версии 4.3.0*

Преобразует заданную строку в знаковое целое число. Если строка начинается с "0x", она интерпретируется как число в системе счисления с основанием 16. Если строка начинается с "0", она интерпретируется как число в системе счисления с основанием 8. Поддерживаются ведущие '+' и '-'.

*Пример: string.to_int("1234") == 1234*

*Пример: string.to_int("-10") == -10*

*Пример: string.to_int("-010") == -8*

### to_int(string, base)

*Добавлен в версии 4.3.0*

Преобразует заданную строку в знаковое целое число, интерпретируя её в указанной системе счисления. Основание должно быть 0 или находиться в диапазоне от 2 до 36 включительно. Если основание равно нулю, строка будет интерпретирована как число с основанием 16, если начинается с "0x", или как число с основанием 8, если начинается с "0". Поддерживаются ведущие '+' и '-'.

*Пример: string.to_int("011", 8) == 9*

*Пример: string.to_int("-011", 0) == -9*

### length(string)

*Добавлен в версии 4.3.0*

Возвращает длину строки, которая может быть любой последовательностью байтов, включая NULL-байты.

*Пример: string.length("AXS\x00ERS") == 7*
