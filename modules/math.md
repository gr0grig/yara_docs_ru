# Модуль Math

*Добавлен в версии 3.3.0*

Модуль Math позволяет вычислять определённые значения из частей файла и создавать сигнатуры на основе этих результатов.

> **Важно:** Где это указано, функции возвращают числа с плавающей точкой. YARA способна автоматически преобразовывать целые числа в числа с плавающей точкой в большинстве операций. Например, 7 будет автоматически преобразовано в 7.0, так как функция entropy возвращает значение с плавающей точкой:
>
> *math.entropy(0, filesize) >= 7*
>
> Исключение — когда функция требует число с плавающей точкой в качестве аргумента. Например, следующее вызовет синтаксическую ошибку, потому что аргументы должны быть числами с плавающей точкой:
>
> *math.in_range(2, 1, 3)*

## Справочник

### entropy(offset, size)

Возвращает энтропию для *size* байтов, начиная с *offset*. При сканировании запущенного процесса аргумент *offset* должен быть виртуальным адресом в адресном пространстве процесса. Возвращаемое значение — число с плавающей точкой.

*Пример: math.entropy(0, filesize) >= 7*

### entropy(string)

Возвращает энтропию для заданной строки.

*Пример: math.entropy("dummy") > 7*

### monte_carlo_pi(offset, size)

Возвращает процентное отклонение от числа Пи для *size* байтов, начиная с *offset*, при проведении теста Монте-Карло для вычисления Пи. При сканировании запущенного процесса аргумент *offset* должен быть виртуальным адресом. Возвращаемое значение — число с плавающей точкой.

*Пример: math.monte_carlo_pi(0, filesize) < 0.07*

### monte_carlo_pi(string)

Возвращает процентное отклонение от числа Пи для заданной строки.

### serial_correlation(offset, size)

Возвращает серийную корреляцию для *size* байтов, начиная с *offset*. При сканировании запущенного процесса аргумент *offset* должен быть виртуальным адресом. Возвращаемое значение — число с плавающей точкой от 0.0 до 1.0.

*Пример: math.serial_correlation(0, filesize) < 0.2*

### serial_correlation(string)

Возвращает серийную корреляцию для заданной строки.

### mean(offset, size)

Возвращает среднее значение для *size* байтов, начиная с *offset*. При сканировании запущенного процесса аргумент *offset* должен быть виртуальным адресом. Возвращаемое значение — число с плавающей точкой.

*Пример: math.mean(0, filesize) < 72.0*

### mean(string)

Возвращает среднее значение для заданной строки.

### deviation(offset, size, mean)

Возвращает отклонение от среднего для *size* байтов, начиная с *offset*. При сканировании запущенного процесса аргумент *offset* должен быть виртуальным адресом. Возвращаемое значение — число с плавающей точкой.

Среднее значение равномерно распределённой случайной выборки байтов — 127.5, которое доступно как константа `math.MEAN_BYTES`.

*Пример: math.deviation(0, filesize, math.MEAN_BYTES) == 64.0*

### deviation(string, mean)

Возвращает отклонение от среднего для заданной строки.

### in_range(test, lower, upper)

Возвращает true, если значение *test* находится между *lower* и *upper* включительно.

*Пример: math.in_range(math.deviation(0, filesize, math.MEAN_BYTES), 63.9, 64.1)*

### max(int, int)

*Добавлен в версии 3.8.0*

Возвращает максимум из двух беззнаковых целых чисел.

### min(int, int)

*Добавлен в версии 3.8.0*

Возвращает минимум из двух беззнаковых целых чисел.

### to_number(bool)

*Добавлен в версии 4.1.0*

Возвращает 0 или 1. Полезно при написании правил на основе баллов.

*Пример: math.to_number(SubRule1) \* 60 + math.to_number(SubRule2) \* 20 + math.to_number(SubRule3) \* 70 > 80*

### abs(int)

*Добавлен в версии 4.2.0*

Возвращает абсолютное значение знакового целого числа.

*Пример: math.abs(@a - @b) == 1*

### count(byte, offset, size)

*Добавлен в версии 4.2.0*

Возвращает количество вхождений указанного байта, начиная с *offset* и просматривая следующие *size* байтов. При сканировании запущенного процесса аргумент *offset* должен быть виртуальным адресом. Параметры *offset* и *size* необязательны; если не указаны, поиск ведётся по всему файлу.

*Пример: math.count(0x4A) >= 10*

*Пример: math.count(0x00, 0, 4) < 2*

### percentage(byte, offset, size)

*Добавлен в версии 4.2.0*

Возвращает частоту вхождения указанного байта, начиная с *offset* и просматривая следующие *size* байтов. При сканировании запущенного процесса аргумент *offset* должен быть виртуальным адресом. Возвращаемое значение — число с плавающей точкой от 0 до 1. Параметры *offset* и *size* необязательны; если не указаны, поиск ведётся по всему файлу.

*Пример: math.percentage(0xFF, filesize-1024, filesize) >= 0.9*

*Пример: math.percentage(0x4A) >= 0.4*

### mode(offset, size)

*Добавлен в версии 4.2.0*

Возвращает наиболее часто встречающийся байт, начиная с *offset* и просматривая следующие *size* байтов. При сканировании запущенного процесса аргумент *offset* должен быть виртуальным адресом. Возвращаемое значение — число с плавающей точкой. Параметры *offset* и *size* необязательны; если не указаны, поиск ведётся по всему файлу.

*Пример: math.mode(0, filesize) == 0xFF*

*Пример: math.mode() == 0x00*

### to_string(int)

*Добавлен в версии 4.3.0*

Преобразует заданное целое число в строку. Примечание: целые числа в YARA — знаковые.

*Пример: math.to_string(10) == "10"*

*Пример: math.to_string(-1) == "-1"*

### to_string(int, base)

*Добавлен в версии 4.3.0*

Преобразует заданное целое число в строку в указанной системе счисления. Поддерживаемые основания: 10, 8 и 16. Примечание: целые числа в YARA — знаковые.

*Пример: math.to_string(32, 16) == "20"*

*Пример: math.to_string(-1, 16) == "ffffffffffffffff"*
