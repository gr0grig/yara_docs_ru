# Модуль Hash

*Добавлен в версии 3.2.0*

Модуль Hash позволяет вычислять хеши (MD5, SHA1, SHA256) из частей файла и создавать сигнатуры на основе этих хешей.

> **Важно:** Этот модуль зависит от библиотеки OpenSSL. Информацию о сборке функций, зависящих от OpenSSL, см. в разделе [Компиляция YARA](../gettingstarted.md). Для пользователей Windows: этот модуль уже включён в официальные бинарные файлы для Windows.

> **Внимание:** Возвращаемая хеш-строка всегда в нижнем регистре. Это значит, что при сравнении хешей в условиях правила `hash.md5(0, filesize) == "feba6c919e3797e7778e8f2e85fa033d"` хеш-строка должна быть задана в нижнем регистре, иначе условие не сработает.

## Справочник

### md5(offset, size)

Возвращает MD5-хеш для *size* байтов, начиная с *offset*. При сканировании запущенного процесса аргумент *offset* должен быть виртуальным адресом в адресном пространстве процесса. Возвращаемая строка всегда в нижнем регистре.

*Пример: hash.md5(0, filesize) == "feba6c919e3797e7778e8f2e85fa033d"*

### md5(string)

Возвращает MD5-хеш для заданной строки.

*Пример: hash.md5("dummy") == "275876e34cf609db118f3d84b799a790"*

### sha1(offset, size)

Возвращает SHA1-хеш для *size* байтов, начиная с *offset*. При сканировании запущенного процесса аргумент *offset* должен быть виртуальным адресом в адресном пространстве процесса. Возвращаемая строка всегда в нижнем регистре.

### sha1(string)

Возвращает SHA1-хеш для заданной строки.

### sha256(offset, size)

Возвращает SHA256-хеш для *size* байтов, начиная с *offset*. При сканировании запущенного процесса аргумент *offset* должен быть виртуальным адресом в адресном пространстве процесса. Возвращаемая строка всегда в нижнем регистре.

### sha256(string)

Возвращает SHA256-хеш для заданной строки.

### checksum32(offset, size)

Возвращает 32-битную контрольную сумму для *size* байтов, начиная с *offset*. Контрольная сумма — это просто сумма всех байтов (беззнаковая).

### checksum32(string)

Возвращает 32-битную контрольную сумму для заданной строки. Контрольная сумма — это просто сумма всех байтов строки (беззнаковая).

### crc32(offset, size)

Возвращает CRC32-контрольную сумму для *size* байтов, начиная с *offset*.

### crc32(string)

Возвращает CRC32-контрольную сумму для заданной строки.
